# 본당 홈페이지 발표자료 작업 기준

> 이 문서는 발표자료(PPT) 제작 시 Claude Code가 따라야 할 작업 기준입니다.
> 모든 슬라이드 작성, 텍스트 생성, 이미지 배치 작업 전에 이 문서를 먼저 참조합니다.

---

## 1. 발표 범위와 정직성

### 1.1 구현된 기능만 다룬다
- 현재 홈페이지가 실제로 수행하는 역할과 기능만 발표 내용에 포함
- 미구현 기능, 향후 계획, 구상 단계 내용은 발표에서 제외
- **예외**: 카카오채널 연동은 포함 가능

### 1.2 이상화된 표현 배제
- 실제로 적용되지 않은 UI/UX 특징(예: "큰 글씨", "단순한 메뉴")은 언급하지 않음
- 사용자가 직접 확인할 수 없는 내부 동작을 과장해서 설명하지 않음

### 1.3 홈페이지의 위치 명확화
- 홈페이지는 본당 사목과 운영의 **보조적 수단**임을 분명히 함
- 본당 운영 전반에 대한 확정적·단정적 표현 사용 금지
- 홈페이지가 본당 운영의 중심처럼 보이게 하는 서술 금지

### 1.4 인사·조직 관련 사안 제외
- 사제, 신부님, 사목회와 관련된 구체적 사안은 다루지 않음
- 인사 이동, 이임 처리, 떠난 수도자 등은 발표에서 제외

## 2. 언어와 표현 규칙

### 2.1 가톨릭 용어 사용 (필수)
| 사용 (○) | 사용 금지 (×) |
|---|---|
| 성전건축 | 성당건축 |
| 공동체 | 우리 |
| (가톨릭 전례·신학 용어 일반) | (개신교식 표현 일체) |

### 2.2 개신교식 표현·용어 절대 금지
- 개신교 교회에서 통용되는 용어, 표현, 어투를 발표 전체에서 사용하지 않음

### 2.3 사용 금지 그룹명
- "우리성당"
- "본당가족"

### 2.4 정확하고 정직한 표현
- 모호한 표현 금지 — 정확하고 분명한 단어 선택
- 거창하거나 자만스럽게 들릴 수 있는 단어·문장 배제
- 자랑·과시로 읽힐 여지가 있는 수사 제거

### 2.5 철학적 서술 자제
- 전체 맥락을 설명하기 위해 꼭 필요한 경우로만 한정
- 기능 설명을 철학적 담론으로 확장하지 않음

## 3. 내용 구성 원칙

### 3.1 전체 구조
- **기승전결**의 흐름으로 구성
- 도입(기) → 전개(승) → 전환·심화(전) → 마무리(결)

### 3.2 핵심 기능 (발표의 중심축)
1. 오늘의 말씀 + 묵상
2. 건축공사일지
3. 내 관심 단체/모임
4. 주보 AI 추출 기능
5. 본당 역사 저장소 역할
6. 보안성

### 3.3 생략할 내용
- 상식 수준의 기초 지식 설명 (인터넷·웹 일반론 등)
- 홈페이지 운영과 직접 관련 없는 기술적 설명
  - 코드 운영 규칙
  - 변경 관리 절차
  - 내부 보안 원칙(운영 측면이 아닌 코드/시스템 차원)
  - 현재 구현 상세

## 4. 시각 자료

### 4.1 단계적 애니메이션이 필요한 두 가지 흐름
두 흐름 모두 이미지를 기반으로 단계별 애니메이션을 적용한다.

**(1) 알림 수신 흐름**
회원가입 → 첫 로그인 → 관심 단체/모임 선택 → 카카오 알림 수신

**(2) 주보 AI 추출 파이프라인**
주보 입력부터 추출 결과까지의 처리 과정을 단계별로 표현

### 4.2 홈페이지 스크린샷 활용
- 실제 홈페이지 화면을 PPT에 사용
- 스크린샷 파일은 별도 폴더에 정리하여 Claude Code에 전달
- 전달 방법은 작업 시작 전 안내된 절차를 따름

## 5. 체크리스트 (슬라이드 작성 시마다 확인)

각 슬라이드를 만들기 전에 다음을 점검한다.

- [ ] 이 슬라이드의 내용은 현재 구현된 기능인가? (카카오채널 제외)
- [ ] 가톨릭 용어를 사용했는가? 개신교식 표현은 없는가?
- [ ] "우리성당", "본당가족"이라는 표현이 들어가 있지 않은가?
- [ ] 거창하거나 자만스러운 표현은 없는가?
- [ ] 모호한 서술 대신 정확한 서술을 사용했는가?
- [ ] 사제·사목회 관련 구체적 사안이 포함되지 않았는가?
- [ ] 홈페이지를 본당 운영의 보조 수단으로 위치시키고 있는가?
- [ ] 상식 수준의 기초 설명이나 불필요한 기술적 설명이 들어가 있지 않은가?
- [ ] 기승전결 흐름 중 어느 위치에 해당하는지 분명한가?
