# 발표자료 스크린샷 폴더

이 폴더에 약속된 파일명으로 스크린샷을 넣어두면, `build_pptx.py` 실행 시 자동으로 슬라이드의 알맞은 자리에 삽입됩니다. 비어 있으면 점선 플레이스홀더가 표시됩니다.

## 필요한 파일 (총 14장)

### 여섯 가지 기능 (승)
| 파일명 | 들어갈 자리 | 어떤 화면을 찍나 |
|--------|-------------|------------------|
| `01-word-meditation.png` | ① 오늘의 말씀 + 묵상 | 홈 첫 화면의 말씀·묵상 카드 |
| `02-construction-log.png` | ② 성전건축 일지 | 성전건축 일지 페이지 |
| `03-member-interests.png` | ③ 내 관심 단체·모임 | 마이페이지의 관심 단체 선택 화면 |
| `04-bulletin-upload.png` | ④ 주보 AI 추출 | 관리자 주보 업로드/추출 결과 화면 |
| `05-archive.png` | ⑤ 본당 역사 저장소 | 주보 아카이브 또는 연도별 목록 |
| `06-kakao.png` | + 카카오 채널 연동 | 카카오 채널 알림 예시 (없으면 비워두기) |

### 알림 수신 흐름 (전)

|--------|-------------|
| `flow1-step1.png` | STEP 1 — 회원가입 화면 |
| `flow1-step2.png` | STEP 2 — 첫 로그인 / 환영 화면 |
| `flow1-step3.png` | STEP 3 — 관심 단체·모임 선택 화면 |
| `flow1-step4.png` | STEP 4 — 카카오 알림 수신 예시 |

### 주보 AI 추출 흐름 (전)
| 파일명 | 들어갈 자리 |
|--------|-------------|
| `flow2-step1.png` | STEP 1 — 관리자 주보 PDF 업로드 화면 |
| `flow2-step2.png` | STEP 2 — AI 분석 진행 화면 |
| `flow2-step3.png` | STEP 3 — 자동 분류 결과 (공지/행사/모임) |

## 권장 사양
- **포맷**: PNG (배경 흰색)
- **해상도**: 가로 **1600~2000px** 권장 (16:9 또는 4:3)
- **민감 정보 제거**: 회원 실명·이메일·전화·주민번호 등 가림 처리

## 빌드 방법
```bash
cd backend && source venv/bin/activate
python ../presentation/pptx/build_pptx.py
```
실행 시 어떤 파일이 들어왔고 어떤 파일이 비어 있는지 콘솔에 표시됩니다.
