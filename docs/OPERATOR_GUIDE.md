# 본당 사무국 운영 가이드

본당 홈페이지를 매일 운영하는 사무국 담당자를 위한 안내서입니다. IT 지식이 없어도 따라할 수 있도록 화면 위주로 설명합니다.

> 시스템 설치·서버 관리는 별도 문서 (`docs/INSTALL.md`) 참고. 이 문서는 설치가 끝난 후 일상 운영 방법.

---

## 0. 처음 사용 시 (1회)

### 0-1. 관리자 로그인

1. 웹 브라우저(크롬 권장)에서 본당 홈페이지를 엽니다.
2. 주소창 끝에 `/admin` 을 붙여 엔터 → 관리자 로그인 화면이 나옵니다. (예: `https://우리본당.kr/admin`)
3. 설치 시 만든 ID·비밀번호로 로그인.

### 0-2. 비밀번호 변경 (강력 권장)

설치 직후 첫 비밀번호는 임시 비밀번호일 수 있습니다.

1. 우측 상단의 본인 이름 클릭 → **마이페이지**
2. **프로필 편집** → 하단의 **비밀번호 변경** 입력 후 저장
3. 영문+숫자 8자 이상, 가능하면 특수문자 포함 추천

### 0-3. 본당 기본 정보 입력

처음 세팅 wizard 에서 본당명·주소를 입력했더라도, 더 자세한 정보를 채울 수 있습니다.

| 메뉴 | 무엇을 입력하나 |
|---|---|
| 성당 정보 → 기본 정보 | 본당명, 주소, 전화/팩스, 네이버 카페·밴드 URL, 지도 좌표 |
| 성당 정보 → 미사 시간 | 요일별 미사 시간 |
| 신부님·수녀님 → 사목자 | 현 주임신부님·보좌신부님·수녀님 사진과 약력 |
| 성당 정보 → 수호 성인 | 본당 수호성인 이름·축일·약력·사진 |
| 레이아웃 → 메인 사진 | 홈 화면 상단 큰 사진 |

## 1. 매주 하는 일

### 1-1. 주일 주보 업로드

가장 자주 하실 일입니다. 주보 PDF 하나 올리면 AI 가 자동으로 공지·행사·말씀을 분리해 줍니다.

1. **주보 관리 → 신규 업로드**
2. **PDF 파일 선택** → 호수·발행일 자동 인식 (수동 수정 가능)
3. **업로드** 클릭 → 자동 분석 시작 (1~3분)
4. 분석 완료 후 **주보 관리 → AI 추출 검토** 에서:
   - 자동으로 공지/행사/모임에 등록된 항목 확인
   - 잘못 분류된 항목은 **이동** 또는 **삭제**
   - 날짜 없는 항목은 **AI 추출(/admin/drafts)** 임시 게시판에 모임
5. 임시 게시판 항목은 검토 후 **게시** 또는 **이동** 처리

> 💡 **팁**: PDF 표지 사진이 자동으로 썸네일이 됩니다. 깨끗한 표지 사진의 PDF 가 좋습니다.

### 1-2. 공지사항 작성

주보 외에 갑작스러운 공지가 있을 때:

1. **소식·일정 → 공지 관리**
2. 우상단 **+ 새 공지** → 제목·본문 입력 → **저장**
3. 사진 첨부도 가능 — 본문 안의 사진 아이콘 클릭

> 📝 **상단 고정**: 중요한 공지는 목록에서 **고정** 체크박스 → 공지 목록 맨 위에 항상 노출.

### 1-3. 본당 일정(행사·모임) 추가

1. **소식·일정 → 본당 일정 (캘린더)**
2. 캘린더에서 **+ 새 일정** 클릭
3. 제목·날짜·구분(행사 또는 모임) 입력 → 저장
4. 자동으로 홈 화면 캘린더, 우측 사이드바 "다가오는 일정" 에 노출

## 2. 자주 하는 일

### 2-1. 회원 관리

새 회원이 가입하면:

1. **게시판·회원 → 회원 관리** 에서 가입자 확인
2. 필요 시 **승인/거부** 또는 **권한 부여** (운영자/관리자)
3. **장기간 미접속 회원**·**탈퇴 회원** 도 같은 화면에서 확인

> 🔒 **민감 정보 주의**: 회원 전화번호·세례명 등은 본당 운영 외 목적으로 사용 금지.

### 2-2. 사진 올리기

행사·전례 사진을 본당 홈페이지에 올리려면:

1. **게시판·회원 → 게시판 관리** 에서 "전례 사진" 또는 "행사 사진" 게시판 선택
2. **+ 글쓰기** → 제목 입력 → 사진 첨부
3. 한 게시글에 사진 여러 장 첨부 가능
4. 자동으로 홈 화면 갤러리에도 표시

### 2-3. 분과·단체 정보 수정

분과나 단체 소개 페이지를 바꾸고 싶을 때:

1. **단체·분과 → 단체·분과**
2. 수정할 분과/단체 선택 → 설명·활동 시간·사진·대표 사진 입력
3. **자세히** 화면(/groups/xxx)에 즉시 반영

### 2-4. 사목자 정보 변경

신부님·수녀님이 새로 오시거나 바뀔 때:

- **현 사목자**: 신부님·수녀님 → 사목자 (현재 본당에 계신 분)
- **떠나신 분**: 신부님·수녀님 → 역대 신부님·수녀님 (자동 이동)
- **본당 출신 사제·수도자**: 신부님·수녀님 → 본당 출신 사제

> 🔄 **자동 이동**: 사목자 화면에서 "역대로 이동" 버튼 → 자동으로 역대 목록에 합류.

## 3. 가끔 하는 일

### 3-1. 백업 다운로드 (월 1회 권장)

데이터를 잃지 않으려면 정기 백업이 필수입니다.

1. **시스템 → 백업** (메뉴 미등록 시 주소창에 `/admin/backup`)
2. **백업 만들기** 클릭 → 1~3분 대기
3. 자동으로 `backup_날짜.tar.gz` 파일이 다운로드됨
4. 다운로드한 파일은 **본당 외부 저장소** (구글 드라이브·네이버 클라우드·USB 등)에 보관

> ⚠️ **중요**: 서버 안에만 보관하면 서버 고장 시 함께 사라집니다. 반드시 본당 외부에 사본 보관.

### 3-2. 사이드바·메뉴 구성 변경

홈 화면 상단 메뉴나 좌측 사이드바를 바꾸려면:

1. **레이아웃 → 메뉴**
2. 메뉴 그룹·항목 추가/이동/삭제
3. 즉시 반영

### 3-3. 메인 페이지(홈) 구성

홈 화면에 어떤 카드를 어떤 순서로 보여줄지:

1. **레이아웃 → 홈 페이지** (블록 빌더)
2. 카드를 위/아래로 드래그하거나 ON/OFF 토글
3. **레이아웃 → 메인 사진**: 상단 큰 사진 슬라이드쇼 관리

## 4. 문제 발생 시

### 4-1. 비밀번호를 잊었을 때

- 회원: 로그인 화면 → **비밀번호 찾기** → 가입 이메일 입력 → 메일 링크로 재설정
- 관리자: 시스템 담당자에게 문의 (별도 절차)

### 4-2. 회원이 "내 정보 다운로드" 또는 "탈퇴" 요청

법적으로 보장된 권리입니다. 회원 본인이 직접:

- **내 정보 다운로드**: 마이페이지 → 프로필 편집 → "내 정보 JSON 다운로드"
- **탈퇴**: 마이페이지 → 프로필 편집 → 하단 "회원 탈퇴"

회원이 절차를 모르면 위 경로 안내해 주세요.

### 4-3. 사이트가 느리거나 접속이 안 될 때

1. 다른 기기·다른 네트워크에서 시도 (본당 와이파이만 문제일 수 있음)
2. 시스템 → 장애 신고 페이지에서 상황 기록
3. 시스템 담당자에게 연락

### 4-4. 잘못된 댓글·게시글 삭제

- 자신의 글: 본인이 직접 삭제 가능
- 다른 회원 글: 관리자 권한으로 게시판 관리 → 해당 글 → 삭제

> ⚖️ **신중히**: 한 번 삭제하면 복구가 어렵습니다. 백업 후 처리 권장.

### 4-5. 외부 키(OAuth·SMTP·AI) 가 만료/변경되었을 때

본당 IT 담당자가 새 키를 발급한 후:

1. **시스템 → 설정** (super-admin 만 접근 가능)
2. 해당 항목 찾아 새 값 입력 → 저장
3. 비밀 항목(시크릿)은 빈 칸으로 두면 기존값 유지

## 5. 자주 묻는 질문

**Q. 주보를 잘못 올렸어요. 다시 분석하고 싶어요.**
→ 주보 관리 목록 → 해당 주보 → **재분석** 버튼. 기존 AI 추출 결과 삭제 후 다시 분석.

**Q. 행사·공지를 작성한 시간이 지났는데 자동으로 안 사라지나요?**
→ 캘린더의 지난 행사는 **기록대기** 상태로 자동 전환됩니다. 사진을 추가해 **기록됨**으로 마무리하면 정리됩니다.

**Q. 메인 화면에 사진이 안 나와요.**
→ **레이아웃 → 메인 사진** 또는 **레이아웃 → 페이지 사진 슬라이드쇼**에서 사진을 등록했는지 확인.

**Q. 본당 이름·로고가 바뀌어요.**
→ **성당 정보 → 기본 정보** 에서 이름 변경 + 로고 업로드. 사이트 전체에 자동 반영.

**Q. 한 줄 봉헌·기도청원은 어떻게 관리하나요?**
→ 일반 게시판과 같이 **게시판·회원 → 게시판 관리** 에서 해당 게시판 선택 후 글 관리.

**Q. 한 게시판을 회원만 볼 수 있게 하려면?**
→ 게시판 관리 → 해당 게시판 → **회원 전용** 체크 → 저장.
